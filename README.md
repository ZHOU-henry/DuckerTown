# Duckertown

Duckertown is Duckermind's fifth top-level project.

It is a pixel-world agent society project:

- a 2D town-like world inspired by the emotional readability of games like
  Stardew Valley
- but populated primarily by AI-agent-controlled characters
- with social memory, roles, routines, relationships, and spatial interaction

The long-term goal is not just a game.

The long-term goal is:

- a large-scale digital society where many independent agents interact inside a
  world with space, time, roles, resources, routines, conflict, and community

## Core Object

Not a chat room.

Not only a social graph.

Not only a simulator dashboard.

Duckertown's core object is:

- `a lived-in town world`

Where:

- agents have bodies, positions, schedules, homes, jobs, and social ties
- humans can observe from a god view or embody one role
- agent society emerges through repeated interaction in a spatial world
